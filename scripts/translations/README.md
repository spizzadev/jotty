# What on earth is this?

Hi, fccview here.
This is a bit of a convoluted way I came up with to make sure translations stay up to date.
Translations are 100% user made and I only add languages that contributors created PRs for.

HOWEVER I am using AI to maintain languages when I create new features, this means that you may see stuff that is not properly translated at times, so please create a pull request with adjustments if needed.

These scripts compare all translation files and allow me to ask AI to create patch files to just update them in batches, so I don't go crazy about it.
